Privacy Policy

Last updated: February 18, 2026

In short: We do not collect, store, or sell any personal data. All processing happens locally in your browser or through the AI provider you choose (Groq or Ollama Cloud).

1 Data We Access

The Extension accesses the following data solely for its core functionality:

  • YouTube Video Transcripts: When you request a summary, the Extension extracts the video's publicly available captions/subtitles from the YouTube page you are viewing.
  • Your API Key: Depending on the provider you choose (Groq or Ollama Cloud), you provide your own API key. It is stored locally in your browser's extension storage (chrome.storage.local) and never leaves your device except for API authentication.
  • Extension Settings: Your preferences (provider choice, language, model, auto-run) are stored locally in your browser.

2 How Data Is Used

  • Transcript text is sent to your chosen AI provider (Groq API or Ollama Cloud API) solely for summarization. The text is processed on the provider's servers and the result is returned to your browser.
  • Your API key is sent only to the provider you selected for authentication. It is never sent anywhere else.
  • No personal information (name, email, browsing history, IP address, etc.) is ever collected or transmitted by us.

3 Data Storage

  • All data is stored locally in your browser using Chrome's extension storage API.
  • Summary results are temporarily cached in your browser to avoid redundant API calls.
  • No data is stored on any external server controlled by us.
  • You can clear all stored data at any time from the Extension's settings ("Clear Cache").

4 Third-Party Services

The Extension communicates with these third-party AI services based on your choice:

  • Groq API (groq.com): Transcript text is sent to Groq's LPU-powered servers for summarization. See Groq's Privacy Policy.
  • Ollama Cloud API (ollama.com): Transcript text is sent to Ollama's cloud infrastructure for summarization. Ollama states they do not log or train on prompts/responses. See Ollama's Terms.

We do not use any analytics, tracking, advertising, or data collection services.

5 Data Sharing

We do not:

  • Sell your data to anyone
  • Share your data with third parties beyond the AI provider you chose
  • Use your data for advertising or profiling
  • Track your browsing activity
  • Collect telemetry or usage analytics

6 Permissions Explained

  • storage: Save your API key and preferences locally in your browser.
  • activeTab: Interact with the YouTube page you're viewing to extract transcript data.
  • Host permission — youtube.com: Run content scripts on YouTube video pages.
  • Host permission — api.groq.com: Send summarization requests to the Groq API.
  • Host permission — ollama.com: Send summarization requests to the Ollama Cloud API.

7 Your Rights

  • Delete all Extension data by removing the Extension from Chrome.
  • Clear cached data from the Extension's settings at any time.
  • Revoke your Groq API key at console.groq.com.
  • Revoke your Ollama API key at ollama.com/settings/keys.

8 Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect any personal information from children.

9 Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Last updated" date above. Continued use of the Extension constitutes acceptance of the updated policy.

10 Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository or contact us through the Chrome Web Store listing.
