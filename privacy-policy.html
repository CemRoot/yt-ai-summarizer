Privacy Policy

Last updated: February 15, 2026

YouTube AI Summarizer ("the Extension") is committed to protecting your privacy. This policy explains what data the Extension accesses, how it is used, and your rights.

Summary: We do not collect, store, or sell any personal data. All processing happens locally in your browser or through the AI API you configure.

1. Data We Access

The Extension accesses the following data solely for its functionality:

  • YouTube Video Transcripts: When you request a summary, the Extension extracts the video's publicly available caption/subtitle data from the YouTube page you are viewing.
  • Your Groq API Key: You provide your own API key, which is stored locally in your browser's extension storage (chrome.storage.local).
  • Extension Settings: Your preferences (language, model choice, auto-run preference) are stored locally in your browser.

2. How Data Is Used

  • Transcript text is sent to the Groq API (api.groq.com) solely for AI summarization. The text is processed by Groq's servers and the summary is returned to your browser.
  • Your API key is sent to Groq API for authentication. It is never sent anywhere else.
  • No personal information (name, email, browsing history, etc.) is ever collected or transmitted.

3. Data Storage

  • All data is stored locally in your browser using Chrome's extension storage API.
  • Summary results may be temporarily cached in your browser session to avoid redundant API calls.
  • No data is stored on any external server controlled by us.
  • You can clear all stored data at any time from the Extension's settings.

4. Third-Party Services

The Extension communicates with the following third-party service:

  • Groq API (groq.com): Used for AI-powered text summarization. Your transcript text is sent to Groq's servers for processing. Please review Groq's Privacy Policy for details on how they handle data.

We do not use any analytics, tracking, advertising, or data collection services.

5. Data Sharing

We do not:

  • Sell your data to anyone
  • Share your data with third parties (other than sending transcripts to Groq for summarization, as described above)
  • Use your data for advertising or profiling
  • Track your browsing activity

6. Permissions Explanation

  • storage: To save your API key and preferences locally in your browser.
  • activeTab: To interact with the YouTube page you're currently viewing and extract transcript data.
  • Host permission (youtube.com): To run the Extension's content script on YouTube video pages.
  • Host permission (api.groq.com): To send API requests for AI summarization.

7. Your Rights

  • You can delete all Extension data by removing the Extension from Chrome.
  • You can clear cached data from the Extension's settings at any time.
  • You can revoke your Groq API key from console.groq.com at any time.

8. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect any personal information from children.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page. Continued use of the Extension after changes constitutes acceptance of the updated policy.

10. Contact

If you have questions about this Privacy Policy, please open an issue on our GitHub repository or contact us through the Chrome Web Store listing.
